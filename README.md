# AddToCart

![Product Screen](./assets/ProductScreen.PNG)
![Cart Screen](./assets/CartScreen.PNG)
Hello! This is a simple Add to Cart app built using React Native and Expo.

## Features

- Add items to cart
- Remove items from cart
- Apply voucher code: discount10 for 10% discount

## Limitations

- No order submission / checkout functionality

## Tech Stack

- React Native
- Expo
- Tailwind CSS

## How to Run

```bash
npm install
npx expo start
```
